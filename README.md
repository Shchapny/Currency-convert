Данное мобильное приложение "Конвертер валют" было разработано во время стажировки в Open Solutions. 
Оно представляет из себя информационный экран, который отображает список (и сохраняем его в БД) с курсами валют от Центрального Банка РФ на текущий день или на выбранную дату. Так же экран для конвертации выбранной валюты в рубли по курсу на выбранную дату.
